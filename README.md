TaskFlow es una web sencilla para la gestión de tareas, pudiendo administrarlas de manera local
Las funciones son:
  - Crear tareas
  - Marcar tareas completadas
  - Eliminar tareas
  - Filtrar tareas
  - Guardar datos de manera local
También tiene creado un modo oscuro y tiene un diseño para móviles
Tecnologías usadas:
  - HTML
  - CSS
  - Tailwind
  - JavaScript
