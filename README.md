TaskFlow es una web sencilla para la gestión de tareas, pudiendo administrarlas de manera local, y en el que tiene creado un modo oscuro y tiene un diseño para móviles.

Las funciones son:
  - Crear tareas
  - Marcar tareas completadas
  - Eliminar tareas
  - Filtrar tareas
  - Guardar datos de manera local

Tecnologías usadas:
  - HTML
  - CSS
  - Tailwind
  - JavaScript
